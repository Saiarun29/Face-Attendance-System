# 🎯 ProAttend – Smart Face Recognition Attendance System

A full-stack **AI-powered attendance system** that uses **face recognition** to automatically mark attendance in real time.

---

## 🚀 Features

- 🔐 Secure Teacher Login
- 📸 Face Registration using Webcam
- 🧠 Model Training (OpenCV - LBPH)
- 🎥 Real-time Face Recognition
- 🗃️ MySQL Database Integration
- 📊 Smart Dashboard:
  - ✅ Present Students
  - 🟡 Registered but Absent
  - 🔴 Not Registered
- 🚫 Prevents duplicate attendance per day

---

## 🛠️ Tech Stack

- Python (Flask)
- OpenCV
- NumPy
- MySQL
- HTML, CSS (Custom UI)

---

## 🧩 System Flow

1. Login as Teacher  
2. Register Student Faces  
3. Train Model  
4. Start Attendance  
5. View Dashboard
